# git_lfs_check

-----------------------

```
usage: git_lfs_check.py [-h] [--root ROOT] [-d] [-q] [--version]

Sanity Check for git lfs extension which verifies that all fils are properly
check into git lfs.

optional arguments:
  -h, --help            show this help message and exit
  --root ROOT, -r ROOT  Path to the root of the repository. Optional argument.
  -d, --debug           Print debug information
  -q, --quiet           Print only errors
  --version             show program's version number and exit
  ```
